# 玻璃厂心理整合游戏设计

## 游戏主题
玻璃厂心灵旅程：以玻璃精灵和工厂为情结，从支持与新鲜感切入，再通过观察、破碎、重组、火焰转化、音乐抚慰，实现心理整合。

---

## 1. 入口：玻璃精灵欢迎页
- 主题：玻璃精灵在工厂门口迎接
- 页面内容：
  - 随机生成一句暖心话或鼓励语
  - “今天的光” / “你此刻最需要的支持”
  - 背景使用轻柔玻璃质感、渐变光线
- 目的：提供新鲜感、支持感、转移注意力，帮助用户从负面情绪中短暂脱离

## 2. 观察区：镜像工厂
- 主题：进入“观察工厂”，用摄像机观察自己
- 功能：
  - 面部表情识别 / 情绪识别
  - 实时镜像 + 情绪标签（平静、悲伤、疲惫等）
  - 视觉反馈：玻璃管中颜色随情绪变化
- 体验：
  - “你现在是什么颜色？”这种拟人化提示
  - 让用户先感知当前状态

## 3. 破碎区：打碎玻璃、表达感受
- 主题：玻璃代表自己的创伤、不良情绪
- 玩法：
  - 点击 / 拖拽选择玻璃块，打碎它
  - 动画表现“碎裂、破碎、裂纹扩散”
  - 同时提供语音输入或文字输入“我现在的感受”
- 结果：
  - 生成一句情绪分析结果（可来自视频分析）
  - 显示“你打破了哪些情绪片段”
- 目的：象征性地承认伤痛、发泄情绪、把负面变成可见素材

## 4. 整合区：拼凑玻璃成新形态
- 主题：把碎片重新组合成别的形状
- 玩法：
  - 拖动多个玻璃碎片进入画布
  - 支持“随意拼凑”和“按提示组合”
  - 可选择不同形状：心、花、翼、光环、星星等
- 交互建议：
  - 每拼一块就有正向提示：“你正在重建”“这是一种新的可能”
  - 让用户自己决定碎片组合方式，强化“我可以重构自己”

## 5. 转化区：火焰重生
- 主题：把拼好的玻璃放进火焰，变成艺术品
- 玩法：
  - 点击“送入熔炉”
  - 动画：玻璃在火中熔化、流动、重组、冷却
  - 最终生成一个新艺术品
- 心理意义：
  - 代表对创伤的整合与重生
  - “旧的破碎被热力融化，新的形态由你创造”

## 6. 音乐情绪回应
- 主题：情绪音乐生成
- 效果：
  - 当用户完成烧制，系统播放由当前情绪分析生成的音乐
  - 音乐跟随“从破碎到安定”的节奏变化
- 体验：
  - 让用户感受到“情绪被理解、被接纳、被抚慰”
  - 结束时显示一句鼓励语或今日总结

## 推荐整体流程
1. 主界面随机暖心语
2. 进入面部/情绪观察工厂
3. 破碎玻璃表达感受
4. 拼凑玻璃重构自己
5. 火中熔化生成艺术品
6. 播放情绪音乐 + 心理整合总结

## 视觉与情感建议
- 画风：温柔梦幻、玻璃光泽、柔和粒子
- 色调：冷暖渐变，破碎时带脆弱感，重组时渐渐明亮
- 文字：鼓励、肯定、温柔陪伴
- 互动节奏：不要太快，控制在节奏缓慢、适度沉浸
