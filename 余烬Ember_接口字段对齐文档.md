## 一、基础信息

| 字段名 | 类型 | 必传 | 格式／范围 | 说明 |
| --- | --- | --- | --- | --- |
| userId | 字符串 | 是 | 登录ID | 用户唯一标识 |
| sessionId | 字符串 | 是 | UUID v4 | 本次记录唯一ID，每次进入流程重新生成 |
| timestamp | 字符串 | 是 | ISO 8601本地时间含时区 | 用户完成输入的时间 |
| inputMode | 字符串 | 是 | video／text／video+text | 用户本次使用的输入方式 |

---

## 二、文字内容

| 字段名 | 类型 | 必传 | 格式／范围 | 说明 |
| --- | --- | --- | --- | --- |
| textInput | 字符串 | 否 | 最多20000字 | 用户手动输入的文字，无则传空 |
| videoTranscript | 字符串 | 否 | 最多20000字 | 视频语音转文字结果，静音或无视频传空 |
| aiSummary | 字符串 | 是 | 最多20000字 | AI综合textInput与videoTranscript提炼的情绪核心描述，作为音乐生成主要文字输入 |

---

## 三、视频分析结果（MediaPipe Face Landmarker）
inputMode = text 时整块传空

| 字段名 | 类型 | 必传 | 格式／范围 | 说明 |
| --- | --- | --- | --- | --- |
| faceDetected | 布尔 | 是 | true／false | 是否检测到人脸 |
| videoQuality | 字符串 | 是 | good／poor／unreadable | unreadable时以下字段均不可信 |
| voiceSpeed | 字符串 | 否 | slow／normal／fast | 语速，静音传空 |
| voiceVolume | 字符串 | 否 | low／normal／high | 音量，静音传空 |
| crying | 布尔 | 否 | true／false | 是否检测到哭腔 |
| sighing | 布尔 | 否 | true／false | 是否检测到叹气 |

---

## 四、情绪综合分析
前半段综合语言与视觉信号处理后输出，语言方向为主权重，强度极低时默认输出平静

| 字段名 | 类型 | 必传 | 格式／范围 | 说明 |
| --- | --- | --- | --- | --- |
| dominantEmotion | 字符串 | 是 | 见情绪枚举表（中文） | 主情绪 |
| overallIntensity | 整数 | 是 | 0－100 | 整体情绪强度，极低时默认平静 |

---

## 五、情绪枚举表
与 MediaPipe Face Landmarker 输出对齐

愤怒／厌恶／恐惧／喜悦／平静／悲伤／惊讶

## 六、视频规格

| 项目 | 规格 |
| --- | --- |
| 支持格式 | mp4 |
| 时长范围 | 3秒－1小时 |
| 文件大小 | 1KB－1GB |
| 弱网／低画质 | 降级为预置视频素材或仅音频分析 |
| 视频分析失败 | 降级为纯文字分析 |
| 静音视频 | 表情分析＋文字分析 |
| 多人画面 | 不处理 |
| 遮脸／背对 | 不处理 |
| 结果传递 | 仅传分析结果，不传视频原文件 |
| 视频存储 | 不存储 |

## 十、触发与流程

| 项目 | 规格 |
| --- | --- |
| 触发方式 | 用户点击按钮，含进度条展示 |
| 触发类型 | 同步 |
| 中途退出 | 后半段不处理，前半段展示结束弹窗 |
| 成长档案跳回重新生成 | 暂不做 |

## 十一、异常与边界

| 项目 | 规格 |
| --- | --- |
| 分析失败 | 默认值兜底 |
| 置信度低 | 默认值兜底，不传置信度标记字段 |
| 输入极短 | 弹窗提示"你说，我在听" |
| 输入极长 | 待定 |
| 同用户多次进入 | 按sessionId区分 |
| 网络中断 | 实时保存，不丢失数据 |
